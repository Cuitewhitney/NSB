Technology

Nextwear Technology utilizes non- invasive, painless method based on frequent ultrasound monitoring combined with AI to create a unique smart clothing system that detect very early cancerous regions of the breast.

Working Principle

A result is created utilizing ultrasounds sent from various sources and sound reflections from the density variation of the cells using embedded miniature echographic transducers in the undergarments. The system provides at-home detection and monitoring of probable malignant cells at an early stage.

Step 1

User wears the bra regularly or when they chose. The bra sends an alert to a smartphone application if the user should wear it more frequently.

Step 2

When the smart bra is on, the embedded sensor sends sound waves to the body.

Step 3

step 3. Result shows on the application.

Analogy

A mammogram uses a low dose of radiation X-ray to take an image of the breast. The tissue is compressed between two plates in order for the best image to be taken. An ultrasound uses high-frequency sound waves and converts them to an image. No radiation is involved.
